# 安装指南（遇依赖报错时按此处理）

正常情况下 `python "${SKILL_DIR}/scripts/setup.py"` 一条命令搞定（自动装 + 脚手架配置）。本文档只在自动安装失败时使用。

## 必需工具

### Windows（主力平台）

```powershell
winget install Gyan.FFmpeg
winget install yt-dlp.yt-dlp
```

- winget 装完**当前终端 PATH 可能未刷新**——脚本会自动扫 `%LOCALAPPDATA%\Microsoft\WinGet\Links\` 与 WinGet Packages 目录兜底，无需重开终端。
- 没有 winget：ffmpeg 从 gyan.dev 下 release 包解压后把 bin 加进 PATH；yt-dlp 用 `pip install --user yt-dlp`。

### macOS

```bash
brew install ffmpeg yt-dlp
```

### Linux

```bash
sudo apt install ffmpeg          # 或 sudo dnf install ffmpeg
pipx install yt-dlp              # 或 pip install --user yt-dlp
sudo apt install fonts-noto-cjk  # 烧中文字幕需要 CJK 字体
```

## 转写引擎（翻译管线必需其一）

**首选：本地 faster-whisper**（零 API 费、词级时间戳）：

```bash
pip install faster-whisper
# 部分 Linux 发行版需要：pip install --break-system-packages faster-whisper
```

首次运行自动从 HuggingFace 下载模型（large-v3-turbo，约 1.6 GB），无需手动准备。CPU 即可跑（int8 量化）。

**兜底：Whisper API**（faster-whisper 装不上时）：

在 `~/.config/video-skills/.env` 里配一个 key：

```
GROQ_API_KEY=...    # 推荐：console.groq.com/keys（便宜、快）
OPENAI_API_KEY=...  # 备选：platform.openai.com/api-keys
```

转写时加 `--engine api`。注意 API 路线按用量计费。

## 输出目录配置

编辑 `~/.config/video-skills/translate.json`：

```json
{
  "output_dir": "C:/Users/<你>/Videos/video-skills-output"
}
```

`output_dir` 必须是绝对路径（或以 `~` 开头）。中间产物写 `tmp/`、最终产物写 `data/`，目录会自动创建。

## 常见问题

| 问题 | 解法 |
|------|------|
| Windows 上 `python3` 跑不了 | Windows 的 `python3` 是 Store 占位符，一律用 `python` |
| winget 装完找不到 ffmpeg | 重开终端，或直接跑脚本（内置 WinGet 路径扫描） |
| faster-whisper 装不上（无编译环境） | 先试 `pip install faster-whisper --only-binary :all:`；仍不行走 API 路线 |
| 烧出的中文字幕是方块 | 缺 CJK 字体：Windows 自带雅黑不会出现；Linux 装 fonts-noto-cjk；macOS 自带苹方 |
| HuggingFace 模型下载慢/失败 | 设 `HF_ENDPOINT=https://hf-mirror.com` 环境变量走镜像 |
